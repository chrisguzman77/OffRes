# DisasterPi Device Usage Guide

## Overview

DisasterPi is a portable emergency device designed to work **completely offline** during disasters. It provides three main features:

1. **AI Assistant** — Ask questions about emergencies (first aid, disasters, safety)
2. **Offline Wallet** — Make payments without internet using QR codes
3. **Offline Map** — Find shelters, hospitals, and aid locations nearby

---

## Getting Started

### Powering On
- Press the power button or connect the power bank
- The device will automatically boot into the main interface
- Wait ~30–60 seconds for startup

### Home Screen
You will see three main options:
- Assistant
- Wallet
- Map

Tap one to begin.

---

## Using the AI Assistant

### What It Does
The assistant provides **life-saving information** during emergencies such as:
- First aid instructions
- Disaster survival steps
- Evacuation guidance
- Water and food safety

### How to Use
1. Tap **Assistant**
2. Type your question (example: *"What should I do during a flood?"*)
3. Press **Ask**
4. Wait for the response (may take a few seconds)

### Tips
- Ask clear, simple questions
- Example questions:
  - "How do I treat a burn?"
  - "What to do in an earthquake?"
  - "Is flood water safe?"

---

## Using the Offline Wallet

### Important
You must **preload money BEFORE losing internet**.

### Check Balance
1. Go to **Wallet**
2. View your current balance

### Make a Payment
1. Tap **Pay**
2. Enter amount and description
3. A QR code will appear
4. Show this QR code to the vendor
5. Vendor scans it to complete the transaction

### How It Works
- Payments are stored locally
- No internet required
- Transactions sync later when connection returns

---

## Using the Offline Map

### What It Shows
- Hospitals
- Emergency shelters
- Water sources
- Aid distribution points
- Safe zones

### How to Use
1. Tap **Map**
2. View nearby locations
3. Tap a point for details

### Tips
- Use this to find help quickly
- Prioritize **hospitals and shelters**

---

## When There Is No Internet

DisasterPi is designed for this situation:
- AI Assistant still works (offline model)
- Wallet still works (QR system)
- Map still works (local data)

---

## Safety Tips

- Always follow official emergency guidance when available
- Use the assistant as **support**, not a replacement for emergency services
- Conserve battery power when possible
- Keep the device dry and protected

---

## Battery and Power

- Use the provided power bank
- Lower brightness to save power
- Avoid running unnecessary features

---

## Troubleshooting

### Device Not Responding
- Restart by reconnecting power

### Screen Not Touching
- Check cable connections

### Slow Responses
- The AI runs locally — delays are normal

---

## Emergency Reminder

If you are in immediate danger:
- Call emergency services if possible
- Move to a safe location
- Use this device to guide your next steps

---

## Summary

DisasterPi helps you:
- **Learn what to do** (Assistant)
- **Buy what you need** (Wallet)
- **Find help nearby** (Map)

It is built to work when everything else fails.
